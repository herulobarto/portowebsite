Python
Cleaning Dataset Car

Proyek yang bertujuan untuk membersihkan dan mempersiapkan dataset mobil menggunakan Anaconda dan pustaka Pandas. Dataset ini berisi informasi tentang mobil, seperti harga, model, tahun, dan lainnya. Proyek ini berfokus pada menangani data yang kosong, mencari data berdasarkan kriteria tertentu, serta menambah nilai data untuk menghasilkan dataset yang bersih dan sesuai kebutuhan user untuk digunakan untuk analisis atau pemodelan lebih lanjut.
(Teknologi : Python, Pandas)

Project Python Cleaning Dataset Car dimulai dengan memuat dataset mobil berformat CSV ke dalam Anaconda menggunakan Pandas. Langkah pertama adalah memeriksa data untuk menemukan nilai null, kemudian mengganti nilai-nilai tersebut dengan rata-rata kolom yang sesuai. Setelah itu, dilakukan analisis distribusi data berdasarkan merk mobil untuk melihat sebaran jumlah mobil per merk. Data juga difilter untuk menampilkan hanya mobil dengan Origin dari Asia dan Eropa, serta jumlah unit lebih dari 400.

Langkah berikutnya adalah melakukan pembaruan pada kolom MPG_City dengan menambahkan nilai 3 pada setiap baris. Proyek ini memastikan dataset bersih, terstruktur dengan baik, dan siap untuk analisis lebih lanjut atau pemodelan prediktif. Dengan menggunakan teknik pembersihan ini, dataset menjadi lebih terorganisir dan dapat memberikan wawasan yang lebih akurat saat digunakan untuk penelitian atau analisis bisnis.
